# 🛍️ Customer Shopping Behavior Analysis

## 📌 Project Overview

This project presents an end-to-end Data Analytics solution focused on understanding customer shopping behavior and uncovering actionable business insights. By analyzing customer purchase data, the project identifies patterns, trends, and opportunities that can help businesses improve decision-making, optimize marketing strategies, and enhance customer satisfaction.

The project demonstrates the complete analytics lifecycle—from data cleaning and exploratory analysis to SQL-based querying and interactive dashboard development.

---

## 🎯 Objectives

* Analyze customer purchasing behavior and spending patterns
* Identify trends across products, categories, and customer segments
* Generate actionable business insights for strategic decision-making
* Build an interactive dashboard for effective data visualization and storytelling

---

## 🛠️ Tools & Technologies

* **Python** – Data Cleaning, Preprocessing, and Exploratory Data Analysis (EDA)
* **MySQL** – Data Extraction, Querying, and Business Analysis
* **Power BI** – Interactive Dashboard Design and Visualization

## 📂 Project Workflow

1. **Data Collection**

   * Imported customer shopping dataset

2. **Data Cleaning & Preprocessing**

   * Handled missing values
   * Removed duplicates
   * Corrected data types
   * Prepared data for analysis

3. **Exploratory Data Analysis (EDA)**

   * Analyzed customer demographics
   * Examined purchasing patterns
   * Identified spending trends
   * Visualized key relationships and distributions

4. **SQL Analysis (MySQL)**

   * Wrote queries to answer business questions
   * Performed aggregations and filtering
   * Extracted meaningful insights from the dataset

5. **Dashboard Development (Power BI)**

   * Built an interactive dashboard
   * Visualized KPIs and trends
   * Enabled dynamic filtering and exploration
